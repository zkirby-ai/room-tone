# Room Tone

Room Tone is a mobile-first PWA for getting conversations going at parties, dinners, mixers, and hangs.

It is not a charisma coach. It is a social momentum tool:
- break awkward openings
- rescue flat 1:1 chats
- widen a group conversation
- use small prompts / games to improve the room's energy

## Current modes
- **Spark** — one good opener
- **Group** — prompts for 3+ people
- **Rescue** — dead-air recovery for 1:1 chats
- **Games** — pass-the-phone / go-around structures

## Development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## PWA
Room Tone includes:
- web manifest
- standalone install behavior
- app icons
- basic service worker shell
